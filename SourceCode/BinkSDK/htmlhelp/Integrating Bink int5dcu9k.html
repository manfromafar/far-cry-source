Integrating Bink into an Nintendo GameCube Build Environment

Discussion

To build an application that uses Bink Video, you must include the Bink SDK in the build process. There are three simple steps to do this: 1) Add the Bink SDK directory to your include and library directories in your project. 2) Include Bink.h in your source files. 3) Link with the BinkNGC.a static library - Bink is directly linked into your application! And that's it - you should now be able to use Bink!
