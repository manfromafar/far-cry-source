Bink SDK 1.5v

Installation for the Xbox

Discussion

The Bink installer will be launched automatically when you insert the Bink SDK CD into your Windows 9x or Windows NT development machine. If auto-play is not enabled on your Win32 machine, you can run the installation program manually by executing BinkXbox.exe in the root directory of the Bink SDK CD.
All installation options are largely automatic, and will not require much further user input unless you would like to change an installation parameter such as the installation directory.
