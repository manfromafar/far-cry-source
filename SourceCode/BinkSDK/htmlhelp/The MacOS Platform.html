The MacOS Platform

Discussion

The second operating system that Bink supported was MacOS. MacOS is strange to develop under - some pieces are advanced, some pieces seem comically backwards. Bink was developed on both MacOS 9 and MacOS X. We supply both a standard MacOS 9 shared library as well as a MacOS X Carbon shared library. Carbon shared libraries work under both MacOS 9 and MacOS X natively - see the Apple developer website for more details. Bink is extremely easy to use and integrate under this platform. This chapter will discuss everything you need to know!
