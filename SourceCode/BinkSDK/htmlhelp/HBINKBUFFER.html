HBINKBUFFER

The basic type HBINKBUFFER is a handle to an opened BinkBuffer.

Discussion

A HBINKBUFFER handle is returned from the BinkBufferOpen function. It also points to a BINKBUFFER structure which contains all sorts of interesting information about the BinkBuffer. If the BinkBufferOpen call failed, then this handle will be zero.
